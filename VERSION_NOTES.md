# Version Notes

## Version 1.0

- โปรแกรมแสดงผลการแข่งขันแบบพื้นฐาน
- แสดงผลการแข่งขันแยกตามประเภทและรอบ
- อัปโหลดไฟล์โดยใช้ชนิดภาพ

## Version 2.0

- เพิ่มส่วนเอกสาร PDF สำหรับผู้ใช้งานทั่วไป
- ผู้ใช้งานทั่วไปสามารถดาวน์โหลดเอกสาร PDF ได้จากหน้าสาธารณะ
- หน้าฝั่งแอดมินรองรับการอัปโหลดและจัดการเอกสาร PDF แยกจากผลการแข่งขัน
