# Version Notes

## Version 1.0

- โปรแกรมแสดงผลการแข่งขันแบบพื้นฐาน
- แสดงผลการแข่งขันแยกตามประเภทและรอบ
- อัปโหลดไฟล์โดยใช้ชนิดภาพ

## Version 2.0

- เพิ่มส่วนเอกสาร PDF สำหรับผู้ใช้งานทั่วไป
- ผู้ใช้งานทั่วไปสามารถดาวน์โหลดเอกสาร PDF จากหน้าสาธารณะได้
- หน้าแอดมินรองรับการอัปโหลดและจัดการเอกสาร PDF แยกจากผลการแข่งขัน

## Version 3.0

- แก้ไขระบบการแสดงผล
- แก้ไขระบบอัปโหลดไฟล์
- กำหนดให้หน้าแรกเป็นหน้าใช้งานสาธารณะ
